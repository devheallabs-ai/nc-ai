# Blog Platform

A blog service where users can create, read, update, and delete posts.
Users need to sign in before creating posts.

## Features
- User authentication (signup, login, logout)
- CRUD operations for blog posts
- AI-powered content suggestions
- Dark theme dashboard

## Pages
- Home page with recent posts
- Dashboard for managing posts
- Login/Signup page
- Post editor page

## API Endpoints
- POST /auth/signup
- POST /auth/login
- GET /posts
- POST /posts
- PUT /posts/:id
- DELETE /posts/:id
- POST /ai/suggest
