# Todo App

A simple todo application for managing daily tasks.
Users can create, complete, and delete tasks organized into lists.

## Features
- Create and manage todo lists
- Add, complete, and delete tasks
- Priority levels (high, medium, low)
- Due date tracking
- Search and filter tasks

## Pages
- Home page with all lists
- List detail page with tasks
- New task form

## API Endpoints
- GET /lists
- POST /lists
- GET /lists/:id
- DELETE /lists/:id
- POST /lists/:id/tasks
- PUT /tasks/:id
- DELETE /tasks/:id
- PUT /tasks/:id/complete
