# NC AI SDK — Code of Conduct

## Purpose

The NC AI SDK is part of the broader NC open-source ecosystem. We want all
discussions, reviews, and contributions to stay respectful, practical, and
welcoming for contributors of all experience levels.

## Expected Behavior

- Be respectful in issues, pull requests, discussions, and chat.
- Give technical feedback on ideas and code — not on people.
- Assume good intent; ask clarifying questions before assuming bad faith.
- Share credit, help unblock others, and keep conversations constructive.
- When disagreeing on technical direction, explain your reasoning with specifics.

## Unacceptable Behavior

- Harassment, hate speech, personal attacks, intimidation, or threats.
- Doxxing, sharing private information, or repeated unwanted contact.
- Sexualized language or imagery in project spaces.
- Deliberate disruption, trolling, bad-faith arguments, or spam.
- Claiming credit for others' contributions.

## Scope

This policy applies across all NC AI SDK community spaces:

- This repository (issues, pull requests, discussions, code review comments)
- GitHub Discussions and related NC project repositories
- Any public or private communication used for project collaboration

## Reporting

Report concerns to: **support@devheallabs.in**

Include: relevant links, screenshots, and timestamps. Reports are handled
confidentially to the extent practical.

## Enforcement

Depending on severity and pattern, responses may include:

1. Private warning and request to change behavior
2. Temporary moderation limits or removal from discussion spaces
3. Permanent ban from project participation

Retaliation against anyone reporting a concern is not allowed.
